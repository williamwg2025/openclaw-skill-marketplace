# Skill Marketplace - 技能市场

发现和安装优质 OpenClaw 技能。

---

## ✨ 功能

- 🛒 **技能发现** - 浏览/搜索技能
- ⭐ **评分评论** - 用户评分和评论
- 📊 **排行榜** - 下载量/评分排行
- 🔔 **更新提醒** - 技能更新通知
- 📦 **一键安装** - 自动下载安装

---

## 🚀 安装

```bash
git clone https://github.com/williamwg2025/openclaw-skill-marketplace.git skill-marketplace
```

---

## 📖 使用

### 浏览技能

```bash
python3 skill-marketplace/scripts/browse.py
```

### 搜索技能

```bash
python3 skill-marketplace/scripts/search.py "backup"
```

### 安装技能

```bash
python3 skill-marketplace/scripts/install.py model-switch
```
